# 💖 Playlist Untukmu

Website playlist romantis interaktif dengan nuansa aesthetic, cerita cinta, timeline kenangan, dan music player modern.

## ✨ Preview

Sebuah website spesial yang dibuat untuk seseorang tersayang.  
Setiap lagu memiliki cerita, kenangan, dan perasaan tersendiri.

---

## 🌸 Features

- 🎵 Interactive Music Playlist
- 💌 Story Mode untuk setiap lagu
- 📅 Timeline Kenangan
- 🌙 Dark / Light Mode
- 💖 Animated Romantic UI
- 🎶 Floating Music Notes Animation
- 📱 Responsive Design
- ✨ Glassmorphism + Gradient Aesthetic
- ❤️ Love Interaction Effects

---

## 🖼️ Tampilan

### Landing Page
- Intro romantis dengan animasi heart pulse

### Playlist
- Daftar lagu aesthetic
- Progress music player
- Mini bottom player

### Story Page
- Cerita spesial di setiap lagu
- Typing animation

### Memories Timeline
- Timeline perjalanan hubungan

### Ending Page
- Romantic closing message
